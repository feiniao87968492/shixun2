# 第一问做题计划：高尔夫球落点影响因素分析

## 1. 任务定义

第一问分为两个子任务。

### 子任务 1：输入参数与输出指标的相关性分析

研究以下输入变量：

* 球速
* 发射角
* 发射方向
* 自旋速率
* 自旋轴偏角
* 后旋
* 侧旋
* 杆头速度
* 攻击角

与以下输出变量之间的统计关联：

* 飞行距离
* 最高点高度
* 总距离
* 横向偏移

输出输入变量与输出变量之间的完整相关性矩阵，并解释主要正相关、负相关和弱相关关系。

### 子任务 2：飞行距离影响因素排序

以飞行距离为目标变量，分别从以下角度衡量输入变量的重要性：

1. 单变量线性关联；
2. 单变量单调关联；
3. 控制其他变量后的线性贡献；
4. 非线性预测贡献；
5. 不同数据处理方案下的排序稳定性。

最终给出“稳定关键因素—次要因素—结论不稳定因素”三级结论，而不是只输出一个没有不确定性说明的排名。

## 2. 核心建模原则

### 2.1 区分边际关联和条件重要性

边际关联回答：

> 单独观察某个变量时，它与飞行距离之间的关系有多强？

使用 Pearson、Spearman 等相关系数衡量。

条件重要性回答：

> 在控制其他输入变量后，该变量还能够提供多少额外信息？

使用标准化岭回归系数和交叉验证置换重要性衡量。

两类结果可能不同，应分别报告，不能混为一个“因果影响”。

### 2.2 避免自旋变量重复表达

自旋速率、自旋轴偏角、后旋和侧旋并不是四个完全独立的信息来源。

相关性分析可以展示全部变量，但多变量模型应分别使用两套自旋表示：

* **表示方案 A：**自旋速率＋自旋轴偏角；
* **表示方案 B：**后旋＋侧旋。

不得在主回归模型中同时使用四个自旋变量。

通过交叉验证误差和排名稳定性比较两种表示方案，选择更适合最终排序的一种；另一种作为敏感性分析。

### 2.3 设置三种样本口径

由于杆头速度和攻击角存在缺失，分别分析：

* **口径 S1：核心变量样本。**不使用杆头速度和攻击角，保留全部可用记录；
* **口径 S2：全变量完整样本。**仅使用九个输入变量均非缺失的记录；
* **口径 S3：全变量插补样本。**对杆头速度和攻击角进行中位数插补，同时增加缺失指示变量。

最终结论优先采用 S1 与 S2、S3 均较稳定的变量。

## 3. 第一阶段：数据审计和预处理

### 3.1 字段检查

检查内容包括：

* 字段名称和数据类型；
* 样本编号是否唯一；
* 重复记录；
* 缺失值数量和比例；
* 无穷值、非数值和异常编码；
* 单位是否与题面一致；
* 每个变量的最小值、最大值、中位数和四分位数。

### 3.2 特殊异常检查

重点检查：

1. 杆头速度为 0 的记录是否为真实测量值或缺失编码；
2. 飞行距离、最高点和横向偏移是否存在明显录入异常；
3. 自旋速率、自旋轴偏角、后旋和侧旋之间是否满足近似几何关系；
4. 球速与杆头速度是否存在明显不合理记录；
5. 极端样本是否对相关系数产生过大影响。

异常记录不直接删除，而是增加质量标记。主分析保留原始记录，敏感性分析再考虑排除极端样本。

生成：

* `q1_data_audit.csv`
* `q1_missing_audit.csv`
* `q1_outlier_flags.csv`
* `golf_shots_clean.csv`

## 4. 第二阶段：相关性分析

### 4.1 Pearson 相关系数

计算输入变量与四个输出指标之间的 Pearson 相关系数：

[
r_{xy}=
\frac{\sum_{i=1}^{n}(x_i-\bar{x})(y_i-\bar{y})}
{\sqrt{\sum_{i=1}^{n}(x_i-\bar{x})^2}
\sqrt{\sum_{i=1}^{n}(y_i-\bar{y})^2}}
]

用于衡量线性关联的方向和强度。

### 4.2 Spearman 秩相关系数

对变量转换为秩后计算相关系数，用于识别单调但不完全线性的关系，并减弱极端值影响。

Pearson 和 Spearman 均作为正文主要结果。

### 4.3 Kendall 秩相关

Kendall 系数作为稳健性验证结果，可以放入附表，不需要作为正文主要排序依据。

### 4.4 相关系数不确定性

采用 Bootstrap 重采样计算主要相关系数的 95% 置信区间。

建议设置：

* Bootstrap 次数：500；
* 随机种子：读取 `configs/default.yaml`；
* 每次有放回抽取原样本规模的数据。

不以显著性检验作为主要判断依据，因为样本量较大时很弱的相关性也可能显著。重点报告相关系数大小、方向和置信区间。

### 4.5 图表

生成：

1. Pearson 输入—输出相关性热力图；
2. Spearman 输入—输出相关性热力图；
3. 飞行距离与排名前四变量的散点图；
4. 散点图叠加 LOWESS 或分箱均值曲线，用于判断非线性关系。

## 5. 第三阶段：飞行距离因素排序

### 5.1 边际相关性排序

分别按照以下指标排序：

[
I_j^{(P)}=|r_j|
]

[
I_j^{(S)}=|\rho_j|
]

其中：

* (r_j) 为变量 (j) 与飞行距离的 Pearson 系数；
* (\rho_j) 为变量 (j) 与飞行距离的 Spearman 系数。

系数正负用于解释影响方向，绝对值用于比较关联强度。

### 5.2 标准化岭回归

对输入变量进行标准化，建立岭回归：

[
\min_{\beta}
\left[
\sum_{i=1}^{n}
\left(
D_i-\beta_0-\sum_{j=1}^{p}\beta_jX_{ij}
\right)^2
+
\lambda\sum_{j=1}^{p}\beta_j^2
\right]
]

使用交叉验证选择正则化参数 (\lambda)。

以标准化系数绝对值作为条件线性重要性：

[
I_j^{(R)}=|\hat{\beta}_j|
]

岭回归用于缓解球速与杆头速度等变量之间的共线性，不采用未经正则化的普通多元线性回归系数作为最终排名依据。

### 5.3 非线性模型

建立一个非线性回归模型，优先选择：

* Extra Trees；
* Random Forest；
* HistGradientBoosting。

通过重复交叉验证比较模型预测误差，选择其中表现稳定且配置较简单的模型。

不直接使用树模型自带的基于节点纯度的重要性，而使用验证集置换重要性：

[
I_j^{(\mathrm{perm})}
=====================

E_{\mathrm{perm}(j)}-E_{\mathrm{original}}
]

其中 (E) 为验证集 RMSE。某变量被随机打乱后误差增加越多，说明该变量提供的预测信息越重要。

### 5.4 重复交叉验证

建议采用：

* 5 折交叉验证；
* 重复 5 次；
* 每次使用不同但可复现的随机种子；
* 数据插补和标准化必须在每个训练折内部完成，防止数据泄漏。

记录每个变量在不同折中的重要性均值、标准差和排名。

## 6. 综合排序方法

不同方法的重要性数值量纲不同，因此不能直接求平均。

对于每种方法 (m)，先将变量重要性转换为名次 (r_{jm})，再转换为标准化得分：

[
s_{jm}=
\frac{p-r_{jm}}{p-1}
]

其中：

* (p) 为参与排序的变量数量；
* 排名第一得分为 1；
* 排名最后得分为 0。

变量 (j) 的综合得分定义为：

[
S_j=
\operatorname{median}
\left(
s_{j,P},
s_{j,S},
s_{j,R},
s_{j,\mathrm{perm}}
\right)
]

使用中位数而非均值，可以避免某一种方法的异常排名主导最终结果。

最终排序表至少包括：

| 字段                     | 含义                 |
| ---------------------- | ------------------ |
| feature                | 输入变量               |
| pearson                | Pearson 系数         |
| spearman               | Spearman 系数        |
| ridge_coef             | 标准化岭回归系数           |
| permutation_importance | 置换重要性              |
| pearson_rank           | Pearson 排名         |
| spearman_rank          | Spearman 排名        |
| ridge_rank             | 岭回归排名              |
| permutation_rank       | 置换重要性排名            |
| aggregate_score        | 综合得分               |
| final_rank             | 最终排名               |
| rank_interval          | Bootstrap 排名区间     |
| top3_frequency         | Bootstrap 中进入前三的比例 |
| direction              | 主要关联方向             |
| stability              | 稳定性等级              |

## 7. 第四阶段：排序稳定性和敏感性分析

至少比较以下场景：

### 7.1 缺失值处理

* 核心变量样本；
* 全变量完整样本；
* 中位数插补＋缺失指示变量。

### 7.2 自旋变量表示

* 自旋速率＋自旋轴偏角；
* 后旋＋侧旋。

### 7.3 异常值处理

* 保留全部样本；
* 排除每个连续变量上下各 1% 的极端记录；
* 对连续变量进行缩尾处理。

### 7.4 模型随机性

* 不同交叉验证划分；
* 不同树模型随机种子；
* 不同置换次数。

### 7.5 稳定性指标

使用：

1. 两种排序之间的 Spearman 排名相关系数；
2. Top-3 和 Top-5 重合率；
3. 每个变量进入前三名的频率；
4. Bootstrap 排名中位数和 95% 区间。

建议定义：

* **稳定关键因素：**多数方法排名靠前，Top-3 频率较高，排名区间较窄；
* **次要因素：**重要性中等，排序随模型略有变化；
* **不稳定因素：**不同样本口径或自旋表示下排名变化明显；
* **弱关联因素：**多数方法中均排名靠后。

## 8. 分组重要性补充分析

为了缓解变量重复表达问题，可增加物理变量组排序：

| 变量组   | 包含变量              |
| ----- | ----------------- |
| 初速度组  | 球速、杆头速度           |
| 发射姿态组 | 发射角、攻击角           |
| 水平方向组 | 发射方向              |
| 自旋状态组 | 自旋速率与自旋轴偏角，或后旋与侧旋 |

通过整组置换计算分组重要性。

分组排名用于回答：

> 飞行距离主要受到速度、发射姿态还是自旋状态控制？

单变量排名用于进一步说明变量组内部哪个指标更重要。

分组重要性作为补充结论，不取代题目要求的单变量排名。

## 9. 论文结论结构

第一问结果按照以下顺序撰写：

1. 数据质量及缺失情况；
2. 输入参数与各输出指标的相关性；
3. 飞行距离的边际相关因素；
4. 控制其他变量后的条件重要性；
5. 综合排名；
6. 缺失值、自旋表示和异常点敏感性；
7. 物理意义解释；
8. 局限性说明。

结论中使用：

* “相关性较强”；
* “提供较高预测信息”；
* “在多种方法中稳定靠前”；
* “可能是飞行距离的重要关联因素”。

避免使用：

* “必然导致”；
* “决定了飞行距离”；
* “提高该变量一定能增加距离”。

## 12. 第一问完成标准

第一问只有同时满足以下条件才能标记为完成：

* 数据审计无未处理的严重错误；
* Pearson 和 Spearman 相关性结果完整；
* 相关系数具有不确定性说明；
* 岭回归和非线性置换重要性均已运行；
* 自旋变量不存在明显重复建模；
* 缺失处理方案经过对照；
* 排名经过 Bootstrap 或重复交叉验证；
* 综合排序公式明确且可复现；
* 关键变量具有统计和物理两方面解释；
* 图、表、生图数据和元数据完整；
* `experiments.md`、`results.md` 和证据链已更新。
