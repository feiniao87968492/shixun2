# 交稿检查表

## 题意与数学逻辑

- [ ] 每个小问都有清晰的数学目标和输出
- [ ] 假设、符号、单位和约束一致
- [ ] 模型选择理由不是“常用”或“高级”
- [ ] 每个主模型都有基线、验证和灵敏度分析

## 数据与复现

- [ ] 原始数据哈希验证通过
- [ ] 数据来源和字段字典完整
- [ ] 依赖环境可安装
- [ ] 从仓库根目录可一键运行
- [ ] 在干净环境完成复现

## 图表与证据

- [ ] 所有论文级图有对应生图数据
- [ ] 所有论文级表有机器可读源文件
- [ ] 图表编号、单位、标题和正文引用一致
- [ ] 关键结论均在证据链中标为 `supported`
- [ ] 失败或被替代结论已保留状态记录

## 论文

- [ ] 摘要数字与最终产物一致
- [ ] 公式变量均有定义
- [ ] 结论不超出证据范围
- [ ] 局限性和推广条件明确
- [ ] 参考文献和数据来源格式统一

## 仓库

- [ ] `python scripts/check_repo.py` 无错误
- [ ] 无密钥、缓存、临时文件和虚拟环境
- [ ] Git 状态清晰，最终提交可定位
