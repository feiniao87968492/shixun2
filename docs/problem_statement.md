# 题目重述与任务拆解

## 赛题背景

本题研究高尔夫球飞行轨迹预测与击球策略优化。题目给定球的质量、直径、重力加速度、海平面空气密度等基础物理参数，并提供一份实测击球数据。需要先从数据中识别影响落点的关键因素，再建立可预测飞行距离和最高点高度的模型，同时构建包含重力、空气阻力和马格努斯力的三维动力学模型，最后在球员可控击球参数范围内寻找使落点接近 200 yd 洞口的最优策略。

## 输入、输出与评价目标

| 小问 | 输入 | 需要输出 | 评价指标 / 判定标准 | 与其他小问的依赖 |
|---|---|---|---|---|
| q1 | Excel 附件中的输入参数：球速、发射角、发射方向、自旋速率、自旋轴偏角、后旋、侧旋、杆头速度、攻击角；输出指标：飞行距离、最高点高度、总距离、横向偏移 | 各输入与输出的相关性矩阵/表；针对飞行距离的影响因素排序；关键与次要因素解释 | 相关系数、秩相关、回归/置换重要性的一致性；不能把相关性表述为因果性 | 无 |
| q2 | q1 清洗后的数据；基础物理参数；球速、发射角、自旋等初始条件 | 飞行距离和最高点高度预测模型；测试集 RMSE 与 MAPE；标定后的 ODE 参数；约 100/150/200 yd 三条典型 3D 轨迹；相对误差分析 | 70/30 随机划分测试误差；ODE 标定误差；典型记录的飞行距离、最高点高度、横向偏移相对误差 | 使用 q1 的数据审计和变量筛选结论 |

## 逐问重述

### q1 影响落点的关键因素分析

- 原题要求：计算各输入参数与输出结果之间的相关性，并针对飞行距离对输入参数影响程度排序。
- 数学化目标：构造输入矩阵 $X$ 与输出矩阵 $Y$，计算线性/秩相关和模型型重要性，输出对飞行距离最重要的变量序列。
- 输出格式：相关性表、热力图、飞行距离影响因素排序表、关键因素解释。
- 题意歧义：球速、杆头速度和攻击角存在缺失或测量链路差异，是否全部纳入同一模型需要由缺失审计决定。
- 采用解释：相关性分析使用全部可用字段；建模型重要性对缺失字段采用显式插补/缺失标记或做完整样本对照。
- 解释的影响：若杆头速度/攻击角缺失较多，其重要性结论需标注样本口径。

### q2 飞行轨迹预测

- 原题要求：建立监督预测模型，计算测试集 RMSE 与 MAPE；建立三维 ODE 模型并标定阻力系数 $C_D$ 和升力系数 $C_L$；选取约 100/150/200 yd 的典型记录绘制轨迹，并与实测指标比较误差。
- 数学化目标：用监督学习估计 $f(X)=(\hat{D}_{carry},\hat{H}_{max})$；用 ODE 计算轨迹 $r(t)=(x(t),y(t),z(t))$，通过最小化实测与仿真指标误差标定 $C_D,C_L$。
- 输出格式：模型性能表、ODE 参数表、三维轨迹图、典型记录误差表、误差原因分析。
- 题意歧义：题面要求 100/150/200 yd，但 OCR 文本中 150 和 200 的数字有识别噪声。
- 采用解释：按飞行距离最接近 100 yd、150 yd、200 yd 的三条记录选样。
- 解释的影响：典型记录选择必须在脚本中固定并记录实际序号，避免手工挑选。

### q3 最优击球策略

- 原题要求：在距离洞口 200 yd 时，使用 q2 模型建立最小化落点到洞口欧氏距离的优化模型，求最优球速、发射角、自旋速率和自旋轴偏角，并绘制最优轨迹。
- 数学化目标：最小化 $J(u)=\sqrt{(D_{carry}(u)-200)^2+L(u)^2}$，其中 $u=(v_0,\theta_0,\omega_0,\phi)$，并满足给定边界约束。
- 输出格式：最优参数表、目标函数值、预测飞行距离与横向偏移、最优 3D 轨迹图、参数灵敏度表。
- 题意歧义：洞口附近是否考虑滚动后的总距离，题面目标表述以落点为主。
- 采用解释：主目标使用首次落地的飞行距离和横向偏移；总距离可作为附加诊断，不进入主目标。
- 解释的影响：论文中需明确“落点接近洞口”而不是“停止点接近洞口”。
