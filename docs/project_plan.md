## 交付顺序

1. 数据审计与清洗：读取 Excel、处理缺失、生成 processed 数据和审计表。
2. q1 基线与主分析：Pearson/Spearman/Kendall 相关性、标准化回归或随机森林/置换重要性对照。
3. q2 监督预测：线性回归基线、随机森林或梯度提升主模型、70/30 测试指标。
4. q2 ODE 建模：建立三维动力学方程、标定 $C_D,C_L$、选择 100/150/200 yd 典型记录。
5. q3 优化：先用 q2 模型做全局搜索/差分进化，再用局部优化精修，并做扰动灵敏度。
6. 论文图表和证据链：每个主张登记脚本、数据、产物和验证结果。

## 里程碑

1. 题意与数据审计完成。
2. 各小问基线完成。
3. 主模型与验证完成。
4. 灵敏度和稳健性完成。
5. 论文图表与证据链闭环。
6. 干净环境复现与交稿检查。
