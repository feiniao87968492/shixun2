# 建模质量契约

本文件定义项目不可降低的质量门槛。

## 结论证据链

每个重要结论必须能定位到输入、脚本、配置、产物和验证。无法回溯的结论不得进入最终论文。

## 数据契约

- 原始数据不可变；
- 字段、单位、缺失值和异常值处理必须记录；
- 外部数据必须记录来源和访问时间；
- 模拟数据必须标记生成机制和随机种子；
- 数据处理步骤必须可由脚本重建。

## 模型契约

- 有基线；
- 有候选模型比较；
- 有参数来源；
- 有验证；
- 有灵敏度或不确定性分析；
- 有适用范围和局限性。

## 产物契约

- 图和生图数据成对保存；
- 论文表格有机器可读源文件；
- 结果文件命名稳定；
- 不手工篡改最终图表数值；
- 关键产物登记到全局索引。

## 复现契约

- 命令从仓库根目录执行；
- 环境依赖可安装；
- 随机性可控；
- 一键运行入口明确；
- 干净环境至少复现一次。
