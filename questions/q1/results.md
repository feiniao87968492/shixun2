# q1 结果与结论

## 1. 最终模型

- 模型：Pearson/Spearman/Kendall 相关性基线 + 标准化岭回归 + ExtraTrees 交叉验证置换重要性 + Bootstrap 排名稳定性。
- 相比基线的改进：不仅给出边际相关性，还给出控制其他变量后的条件线性贡献、非线性预测贡献和不同样本口径/自旋表示下的稳定性。
- 关键参数：随机种子 2026；Bootstrap 500 次；5 折交叉验证重复 5 次；置换重要性每折重复 5 次。

## 3. 验证结果

- 验证方法：Pearson/Spearman/Kendall 相关性、Bootstrap 相关系数置信区间、岭回归条件贡献、ExtraTrees 交叉验证置换重要性、缺失处理和自旋表示敏感性对照。
- 验证指标：相关系数、95% Bootstrap 区间、标准化岭回归系数、置换后 RMSE 增量、Top-3 频率、排名区间、分组置换重要性。
- 是否通过：通过。`python questions/q1/scripts/validate.py` 检查 26 个表格/图片/图数据/元数据产物全部存在。
- 主要误差或失败案例：发射角、自旋轴偏角、侧旋、后旋等变量在边际相关和条件重要性之间存在差异，说明它们的排序对模型口径更敏感；杆头速度和攻击角因缺失需要保留完整样本与插补样本对照。

## 4. 灵敏度与稳健性

- 关键扰动：核心变量样本 S1、全变量完整样本 S2、全变量插补样本 S3、自旋率/轴角表示、自旋后旋/侧旋表示、剔除 1% 极端值、1% 缩尾。
- 结果变化：球速在所有方法中稳定第一；杆头速度边际相关强但在控制球速后条件贡献变弱；攻击角进入前三的频率为 0.85；发射角和自旋轴偏角在模型型指标中重要，但 Bootstrap 边际排名区间较宽。
- 稳健性结论：稳定关键因素为球速、杆头速度、攻击角；次要或口径敏感因素包括发射角、自旋轴偏角和侧旋；弱关联因素包括后旋、发射方向和自旋速率。

## 5. 可写入论文的结论

在 735 条实测记录中，球速与飞行距离的关联最稳定，Pearson 相关系数为 0.758，Spearman 相关系数为 0.776，Bootstrap 排名区间为 1-1，是第一问中最可靠的关键因素。杆头速度与飞行距离也呈较强正相关（Pearson=0.469，Spearman=0.583），但在控制球速后条件贡献明显减弱，说明它与球速存在信息重叠。攻击角的边际相关较弱但排序稳定性较高，发射角和自旋轴偏角更多体现为条件模型和非线性模型中的补充信息。整体看，速度组对飞行距离的预测信息最高，发射姿态组次之，自旋状态组中等，水平方向组对飞行距离本身的贡献最弱。

以上结论是统计关联和预测信息结论，不等同于因果效应。

## 6. 局限与适用范围

- 杆头速度和攻击角分别有 63 和 65 条缺失，相关结论依赖插补与完整样本对照。
- 自旋速率/自旋轴偏角与后旋/侧旋存在重复表达，主模型没有同时使用四个自旋变量；后旋和侧旋用于敏感性对照。
- 排名反映当前数据集内的统计关联和模型预测贡献，不能直接推出改变某变量必然增加飞行距离。
- 部分变量的条件重要性受共线性影响，需要在第二问预测模型中继续诊断。
